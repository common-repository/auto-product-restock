A simple way to automatically restock products in WooCommerce daily.

== Description ==
Never forget to restock your products in WooCommerce! Automatically restock individual product quantity amounts in WooCommerce on a daily basis. Set amounts and times for each product you want to restock.

Start using the plugin by navigating to the WooCommerce product's inventory tab. Be sure that "Manage Stock?" is checked. Three new fields that control the restock features will appear.

== Installation ==
1. Upload and extract the .zip file downloaded to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using the plugin
